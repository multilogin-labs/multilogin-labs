# Code of Conduct

We follow the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## In short

- Be respectful. No harassment, hate speech, or personal attacks.
- Stay on-topic: Multilogin, antidetect browsers, cloud phones, MMO automation, partner content.
- No spam, link farming, or unrelated affiliate links (this repo's only affiliate is the Multilogin partner link).
- Reports: open a private report via GitHub Security Advisories or contact a maintainer.

## Enforcement

Maintainers may remove comments, close PRs/issues, or ban accounts that violate the spirit of this Code.

This applies to all repository spaces: issues, PRs, discussions, code review.
