# Recipe: Clone profile for scaling

`POST https://api.multilogin.com/profile/clone`

Duplicate an existing warmed profile to new identities (new profile IDs) — faster than manual create.

See full body parameters in [profile-clone.md](../endpoints/profile-clone.md).

**Tip:** Clone after proxy + fingerprint template is proven — tag clones by batch date in dashboard.
