# Hướng dẫn Multilogin API tiếng Việt

> [Giá đối tác →](https://multilogin.com/pricing/?utm_source=saas&utm_medium=partner&a_aid=saas&a_bid=f5fad549) · **`SAAS50`** · **`MIN50`**

API Multilogin X có 90 endpoint, chia 2 phần: Cloud (`api.multilogin.com`) và Launcher (`launcher.mlx.yt:45001`).

## Các bước

1. Lấy token: [authentication](../../api/authentication.md).
2. Start profile: `GET /api/v2/profile/f/{folder_id}/p/{profile_id}/start`.
3. Stop profile: `GET /api/v1/profile/stop?profile_id=...`.
4. Khám phá API: [Swagger UI](../../api/swagger.html), [Cheatsheet](../../api/CHEATSHEET.md), [Cookbook](../../api/cookbook/).
