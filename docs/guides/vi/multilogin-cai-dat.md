# Cài đặt Multilogin trên Windows / macOS / Linux

> [Giá đối tác →](https://multilogin.com/pricing/?utm_source=saas&utm_medium=partner&a_aid=saas&a_bid=f5fad549) · **`SAAS50`** · **`MIN50`**

Tải app từ trang chính thức, đăng nhập bằng tài khoản partner ($2 trial), kiểm tra Launcher chạy ở `https://launcher.mlx.yt:45001`.

## Các bước

1. Windows: tải `.exe`, cài bình thường, mở app, login.
2. macOS: `.dmg`, cấp quyền Accessibility nếu app yêu cầu.
3. Linux: hỗ trợ Ubuntu/Debian; có thể chạy trên VPS với GUI hoặc Xvfb.
4. Sau khi login, mở [docs/api/quick-start](../../api/quick-start.md) để lấy token.
