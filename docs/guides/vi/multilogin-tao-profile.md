# Cách tạo profile trình duyệt antidetect

> [Giá đối tác →](https://multilogin.com/pricing/?utm_source=saas&utm_medium=partner&a_aid=saas&a_bid=f5fad549) · **`SAAS50`** · **`MIN50`**

Profile = một danh tính trình duyệt độc lập (fingerprint, cookies, proxy, ngôn ngữ).

## Các bước

1. Mở app → Folders → tạo folder theo client/dự án.
2. New profile → chọn Mimic (Chrome) hoặc Stealthfox (Firefox).
3. Gán proxy phù hợp với quốc gia mục tiêu.
4. Đặt tag (`tiktok`, `warmup`, `client-x`) để dễ search và filter API.
5. Quick profile dùng cho phiên một lần — xem [cookbook 02](../../api/cookbook/02-quick-profile-scrape.md).
