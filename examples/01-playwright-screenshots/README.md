# Example 01 · Playwright screenshot grid

Connect to a running Multilogin profile and screenshot a list of URLs.

## Run

```bash
export MULTILOGIN_TOKEN=...
export MULTILOGIN_FOLDER_ID=...
export MULTILOGIN_PROFILE_ID=...
node screenshot-grid.mjs
```
