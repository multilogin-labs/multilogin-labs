# examples · runnable mini-projects

Each folder is a small, self-contained project showing one Multilogin Labs pattern. Copy a folder into your own repo and adapt.

| Example | Stack | Description |
|---|---|---|
| [01-playwright-screenshots](01-playwright-screenshots/) | Node + Playwright | Connect to a Multilogin profile and capture a screenshot grid. |
| [02-python-bulk-create](02-python-bulk-create/) | Python | Read a CSV and create one Multilogin profile per row. |
| [03-warmup-cron](03-warmup-cron/) | Bash + cron | Schedule the warmup scheduler from cron. |
| [04-discord-alerts](04-discord-alerts/) | Node | Webhook alert when a profile fails to start. |

Each example links back to the relevant cookbook recipe in [`docs/api/cookbook/`](../docs/api/cookbook/README.md).
