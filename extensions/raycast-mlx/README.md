# raycast-mlx · Raycast extension scaffold

Quickly search Multilogin X API endpoints from Raycast on macOS.

## Status

**Scaffold** — `npm install && npm run dev` to test inside Raycast.

## Files

- `package.json` — Raycast manifest
- `src/search-endpoints.tsx` — list view with fuzzy matching

## Dev

```bash
cd extensions/raycast-mlx
npm install
npm run dev
```

Then trigger Raycast and run "Search Multilogin endpoints".

PR welcome to publish on raycast.com/store.
