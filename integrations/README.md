# integrations · third-party glue

| Integration | What |
|---|---|
| [cloudflare-worker](cloudflare-worker/) | Edge proxy: search, partner redirect, doc bouncer |
| [telegram-bot](telegram-bot/) | `/search`, `/trial`, `/cheatsheet`, `/plan` |
| [discord-bot](discord-bot/) | `!mlx search …`, `!mlx trial`, `!mlx cheatsheet`, `!mlx plan` |
| [wordpress-plugin](wordpress-plugin/) | Shortcode `[mlx_partner]` for affiliate banner |

> All integrations consume the same `endpoints.json` and partner URL. No vendor lock-in beyond the Multilogin partner relationship.
