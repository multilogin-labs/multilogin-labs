# mlx-discord-bot

```bash
cd integrations/discord-bot
npm install
DISCORD_BOT_TOKEN=... npm start
```

## Commands

- `!mlx help` · `!mlx search <terms>` · `!mlx trial` · `!mlx cheatsheet` · `!mlx plan`

## Required intents

- Guild Messages, Message Content (enable in the Discord developer portal).

## Deploy

- Fly.io, Railway, Render, or self-hosted Docker.
